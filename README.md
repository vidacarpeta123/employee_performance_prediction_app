# Employee Performance Prediction App

This Streamlit web application predicts employee performance levels (Good, Excellent, Outstanding) using a machine learning model trained on historical employee data. The app is designed to assist HR professionals and managers in making informed decisions about employee development, recruitment, and recognition.

# Features:

- Input employee information through an interactive form
- Predict performance level and view model confidence
- Analyze feature importance for each prediction
- Explore EDA visualizations for insights into the dataset
- Tabbed interface for information, predictions, data exploration, and contact details

# Model Inputs:

- Age, Gender, Education Background, Marital Status
- Department, Job Role, Business Travel Frequency
- Distance From Home, Education Level, Environment Satisfaction
- Hourly Rate, Job Involvement, Job Level, Job Satisfaction
- Number of Companies Worked, Over Time, Last Salary Hike Percent
- Relationship Satisfaction, Total Work Experience, Training Times Last Year
- Work Life Balance, Experience Years at Company, Experience Years in Current Role
- Years Since Last Promotion, Years With Current Manager, Attrition


# Model Performance:

- Accuracy: 93.33%
- Precision: 93.34%
- Recall: 93.33%
- F1 Score: 93.33%
- ROC AUC: 96.76%

# Usage:

- Clone the repository and ensure employee_performance_model_.pkl is in the project directory.
- Run streamlit_app.py using Streamlit:
- Fill in employee details and click "Predict Performance" to view results.
